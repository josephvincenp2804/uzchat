# uzchat

A simple peer-to-peer chat application using WebRTC technology.

## Features
- Direct browser-to-browser connection
- No server required for messaging
- Manual signaling for connection setup
- Clean, responsive UI

## How to Use
1. Open the hosted page
2. Create a room or join with a code
3. Share signaling data with your peer
4. Start chatting!
